# 更新日志

## [1.0.0] - 2026-03-13

### 新增功能
- 支持 Excel 和 CSV 格式文件的导入和导出
- 灵活的列名映射配置
- 智能日期格式识别和转换 (支持 yyyymmdd, yymdd, yymmd 格式)
- 自动补全交易金额正负号
- 支持多 Sheet 页面选择
- 自定义表头行位置
- 数据清洗预览功能
- 实时清洗统计信息

### 技术特性
- 跨平台支持 (Windows, macOS, Linux)
- Electron + Python 混合架构
- GPL-3.0 开源协议
- GitHub Actions 自动化构建
