===========================================
🔑 Personal Access Token 获取
===========================================

如果你还没有 Token:

1. 访问: https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 填写信息:
   - Note: table-cleaner
   - Expiration: 90 days 或 No expiration
   - Select scopes: 勾选 ✅ repo
4. 点击 "Generate token"
5. 复制生成的 Token (只显示一次!)
6. 粘贴到 Git 提示的 Password 处

⚠️ 注意: Token 不是你的 GitHub 密码!

❓ 常见问题
===========================================

Q: 推送时提示 "fatal: Authentication failed"
A: 检查 Token 是否正确,确保有 'repo' 权限

Q: 推送时提示 "Updates were rejected"
A: 使用强制推送: git push origin master --force

Q: 找不到 Git 命令
A: 安装 Git: https://git-scm.com/downloads

Q: 不想用命令行
A: 使用 GitHub Desktop 或 VS Code 的 Git 集成
